# Super-Store-Sales-Dashboard-project
Super Store Sales Dashboard is an interactive Excel-based data analytics project designed to analyze sales performance. It tracks KPIs like Total Sales, Transactions, Quantity, and Average Sales, and provides insights on regional trends, top cities, customer segments, and shipping modes to support data-driven decisions.
